# Ntegra technical test

## How to Run

First, run 

```npm install```

in your terminal. 

Make sure you're working in the correct directory, and that the start script is set up in your package.json file. This should look like:

``` 
"scripts": {
    "start": "react-scripts start",
    ...
  }, 
  ```

Then, run 

``` npm run start ```

in your terminal.
